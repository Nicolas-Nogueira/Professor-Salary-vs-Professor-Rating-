# Comparing-Prof-Salary-to-their-Rating-

This project analyzes the relationship between professors' salaries and their ratings. The data is sourced from Boise State University (BSU) and includes salary and rating information for professors.

## Project Structure

- **BSU_prof_salary.csv**: Raw data in CSV format containing professors' salaries and ratings.
- **BSU_prof_salary.xlsx**: Raw data in Excel format containing professors' salaries and ratings.
- **data_analysis.ipynb**: Jupyter Notebook for analyzing the data and visualizing the relationship between salaries and ratings.
- **data_scrapping.ipynb**: Jupyter Notebook for scraping or collecting the data used in this project.
- **README.md**: Project documentation.

## Data Analysis

The analysis includes:

- Statistical summaries of salaries and ratings.
- Visualizations to identify trends and correlations.
- Insights into how ratings impact salaries.
